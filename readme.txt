== Installation ==

= Minimum Requirements =

* WordPress 6.7 or greater
* WooCommerce 9.0 or greater
* PHP version 8.1 or greater
* MySQL version 5.6 or greater
* WP Memory limit of 128 MB or greater (256 MB or higher is preferred)

= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of plugin_name, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “plugin_name” and click Search Plugins. Once you’ve found our eCommerce plugin you can view details about it such as the the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Manual installation =

The manual installation method involves downloading and uploading our plugin to your webserver via your favorite FTP application.
